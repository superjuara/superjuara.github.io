                            <p>Kalau Anda ingin menjadi seorang web developer yang dapat mendesain maupun melakukan pemrogramman sekaligus, maka JavaScript merupakan bahasa pemrogramman wajib yang harus dikuasai. JavaScript adalah bahasa pemrogramman yang berkembang pesat, yuk kita kenalan dengan javascript.</p>
<p><strong>Karakteristik JavaScript</strong></p>
<p>Beberapa karakteristik dari JavaScript sebagai berikut:</p>
<ul>
<li>
<p>Bahasa pemrogramman berjenis high-level programming (struktur dab syntaxnya mudah dipahami karena menggunakan bahasa yang mudah dimengerti manusia).</p>
</li>
<li>
<p>Bersifat client-side (hanya membutuhkan web browser untuk menguji JavaScript)</p>
</li>
<li>
<p>Berorientasi object (cocok buat Anda yang ingin masuk ke dalam konsep pemrogramman berbasis object).</p>
</li>
<li>Bersifat loosely typed (tidak membutuhkan deklarasi variable terlebih dahulu).</li>
</ul>
<p><strong>Tool dan Syarat yang Dibutuhkan</strong></p>
<p>Unuk mempelajari dan membuat pemrogramman website menggunakan JavaScript, Anda tidak perlu membutuhkan tool khusus. Minimal Anda membutuhkan 2 tool standart berikut:</p>
<ul>
<li>
<p><strong>Text-Editor</strong> : silahkan gunakan Text-Editor seperti Notepad, Notepad ++, Sublime Text, VScode, Atom, dll. sesuiakan dengan selera Anda.</p>
</li>
<li>
<p><strong>Browser</strong> : Anda bisa menggunakan Browser sesuai selera, seperti Mozilla Firefox, Google Chrome, Safari, Internet Explorer, dll.</p>
</li>
<li><strong>HTML</strong> : Sebelum belajar JavaScript, sebaiknya Anda sudah paham dasar-dasar tentang HTML terlebih dahulu, karena JavaScript ditulis diantara tag-tag HTML.</li>
</ul>
<p><strong>Fungsi JavaScript dalam Desain Website</strong></p>
<p>Pemrogramman dasar untuk pembuatan website adalah HTML. Namun HTML hanya bisa menghasilkan dokumen statis dan non interaktif. JavaScript diciptakan untuk mengatasi kelemahan ini. </p>
<p>Apabila Anda ingin membuat situs yang dapat berinteraksi antara user dengan website secara cepat tanpa harus melibatkan web server, maka maka digunakanlah JavaScript. </p>
<p>Tanpa koneksi internet sekalipun, data yang diinput oleh user dapat diproses langsung. Hal ini terjadi karena JavaScript bersifat client-side.</p>
<p><strong>Kelemahan JavaScript</strong></p>
<p>Dibalik kehandalan dan popularitasnya, JavaScript juga memiliki kelemahan-kelemahan. berikut ini kelemahan yang ada di JavaScript:</p>
<ul>
<li>
<p>Pengaksesan internet dapat mematikan fitur JavaScript pada browser sehingga ada kemungkinan website itu menjadi menjadi tidak berjalan semestinya bagi sisi pengakses.</p>
</li>
<li>
<p>JavaScript dianggap pemrogramman yang tidak aman karena kodenya pemrogrammanya dapat dilihat dan dimodifikasi oleh pengakses website.</p>
</li>
<li>Sangat tergantung pada kapabilitas browser terhadap kode-kode JavaScript. Itu artinya, pemrogramman JavaScript saat ini mungkin tidak di support oleh browser versi lama.</li>
</ul>
<p><strong>Mencoba JavaScript Pertama “Hallo Dunia”</strong></p>
<p>Sudahkah siap membuat website dengan JavaScript ?, yuk kita coba langsung</p>
<p>Buka <strong>Text-Editor</strong></p>
<p>Kemudian masukkan dan simpan kode berikut ini dengan extensi <strong>.html</strong>, contoh <strong>hallo.html</strong>:</p>
<pre><code>&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
    &lt;title&gt;Hallo Dunia! - Belajar JavaScript Pertama&lt;/title&gt;
    &lt;script&gt;
        function halodunia()
        {
            var halo = document.getElementById("container");
            halo.innerHTML+="&lt;p&gt; Belajar JavaScript di RubyPedia.com&lt;/p&gt;";
        }
    &lt;/script&gt;
&lt;/head&gt;
&lt;body&gt;

    &lt;h1&gt;Belajar JavaScript&lt;/h1&gt;

    &lt;!-- klik fungsi hallo dunia() --&gt;
    &lt;button id="klik" onclick="halodunia()"&gt;Tekan Aku&lt;/button&gt;

    &lt;!-- menampilkan pesan Belajar JavaScript di RubyPedia.com --&gt;
    &lt;div id="container"&gt;&lt;/div&gt;

&lt;/body&gt;
&lt;/html&gt;</code></pre>
